开启本地服务器

发表于 2022-02-11 更新于 2022-03-03 分类于 前端开发 阅读次数：

安装拓展“Live Server”

打开vs code，在拓展处搜索“Live Server”并安装

打开项目

1、从“文件”————“打开文件夹”来打开项目

2、或者直接将项目文件夹拖到“vs code”

点击右下角的“Go Live”

在开启的服务器后，在文件资源管理器中，选择要预览的 .html 文件，右键选择 以 live server 方式打开。
在打开的浏览器中即可预览项目
